Simple Calculator

A clean and beginner-friendly calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a modern, responsive user interface.
Features
Addition (+)
Subtraction (-)
Multiplication (×)
Division (÷)
Decimal number support
Clear display (C)
Delete last character (⌫)
Error handling for invalid expressions
Simple and responsive design
🛠️ Technologies Used
HTML5 – Structure of the calculator
CSS3 – Styling and layout
JavaScript – Calculator functionality and logic

How to Run
Download or clone the project.
Open the index.html file in any modern web browser.
Start performing calculations.

User Interface
Clean card-style calculator layout
Soft blue color palette
Grid-based button arrangement
Hover effects for better user experience

Functionality
Button	Function
0–9	Enter numbers
.	Decimal point
+, -, ×, ÷	Arithmetic operations
=	Calculate result
C	Clear display
Delete last character

How It Works
User inputs numbers and operators using buttons.
The expression is displayed on the screen.
Pressing = evaluates the expression and shows the result.
Invalid expressions display an Error message.
After calculation, entering a new value starts a fresh expression.
